About

Engineer by training, builder by obsession.

I'm Rushal Vaghani, a senior full-stack engineer based in Surat, India, working remotely with founders, agencies, and product teams across the US, EU, and Asia.

For over two years, I've shipped production software end-to-end — from greenfield SaaS architecture to AI-powered features in mature codebases. My core stack is TypeScript, React, Node.js, Next.js, React Native, and Python, with growing expertise in Rust and AI infrastructure.

I care deeply about craft. Clean code is non-negotiable, but so is shipping. I optimize for the team that inherits the codebase — readable abstractions, sensible defaults, and tests that catch real regressions.

When I'm not coding, I'm sharpening competitive programming skills (rated 1808 on CodeChef), exploring new languages, and reading systems engineering literature.
